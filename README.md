# Task 6 - Portfolio Website with Flask

## Objective
Build a personal portfolio website using Flask.

## Technologies Used
- Python
- Flask
- HTML
- CSS

## Features
- Flask routing
- Portfolio homepage
- About section
- Projects section
- Contact form

## Project Structure
- app.py → Flask backend
- templates/index.html → main webpage
- static/style.css → styling

## How to Run
1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

## Outcome
Created a basic portfolio website using Flask with routing and form handling.
